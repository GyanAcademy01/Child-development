# 📐 Task Summary Format

## Standard Format (Small Task — 1-3 files)
```markdown
✅ કામ પૂર્ણ:
1. `file.ext` — શું બદલ્યું (1 line)
2. `file.ext` — શું બદલ્યું (1 line)

📊 **File Edit Stats:**
- Edited `file1.ext` +X -Y
```

## Table Format (Large Task — 4+ files)
```markdown
## ✅ કામ પૂર્ણ

| # | File | Change |
|:-:|------|--------|
| 1 | `file.ext` | Short description |
| 2 | `file.ext` | Short description |

**Status:** ✅ Complete

📊 **File Edit Stats:**
- Edited `file1.ext` +X -Y
- Edited `file2.ext` +X -Y
```

## Rules
- ગુજરાતી only — English file names ચાલશે
- Numbered list or Table format — no long paragraphs
- Small task (1-3 files) → numbered list
- Large task (4+ files) → table format
- ❌ બિનજરૂરી explanation ન આપવી
- ❌ Task પહેલાની detail repeat ન કરવી
- ✅ Short, clear, numbered
- ✅ Task ને improve કરવા માટે જો કોઈ suggestions હોય તો છેલ્લે ઉમેરવા
- ✅ છેલ્લે કઈ ફાઇલમાં કેટલી લાઇન ઉમેરાઈ (+) અને કેટલી દૂર થઈ (-) તેની માહિતી આપવી (દા.ત. `Edited filename.ext +10 -5`)
